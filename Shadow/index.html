Shading can add alot to the design of a webpage by adding depth and an overall more immersive feel


this design can be implemented in CSS or HTML through the element "box-shadow"


The box shadow element will look something like this: "box-shadow: right(px) top(px) (blur size) rgba(red, greem blue, alpha)"
